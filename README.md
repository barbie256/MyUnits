# MyUnits

A property management application built with Python to help landlords manage properties, units, tenants, rent payments, and receipts.

## Features

- Property management
- Unit management
- Tenant management
- Rent payment tracking
- Receipt generation
- Property dashboards

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic

## Project Status

This project was built as part of my software development learning journey and property management platform development.
