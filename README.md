# MyUnits

A property management application built with Python to help landlords manage properties, units, tenants, rent payments, and receipts.

## Features

- Property management
- Unit management
- Tenant management
- Rent payment tracking
- Receipt generation
- Property dashboards

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic

## Project Status

This project was built to help landlords and property managers manage properties, tenants, rent payments, and receipts more efficiently.
